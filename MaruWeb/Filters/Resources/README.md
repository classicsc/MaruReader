This directory contains MPL-2.0 adblock runtime resources used by MaruWeb
cosmetic filtering.

- `maru-content-cosmetic-ios.js` is a bundled copy of Brave's
  `content_cosmetic_ios.js` and `procedural_filters.ts` runtime.
- `adblock-resources.json` is the generated resource table from
  `brave/adblock-resources`, including uBO-compatible scriptlets for
  adblock-rust.

Sources:

- https://github.com/brave/brave-core
- https://github.com/brave/adblock-resources
