# MaruReader Privacy Statement (Updated April 2026)

MaruReader does not collect personal data of any kind. Your data stays on your device by default.

If you use the dictionary update function, a network request will be sent to the update URL defined in each dictionary's manifest, if any. The operator of the update server can see your IP address and know that you are using MaruReader with that dictionary.

Some information is sent over the network to the servers you configure if you use a custom audio server or Anki-Connect. To protect your privacy, only use these features with your own servers.
